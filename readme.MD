
## 🎯 Temas Desarrollados

A lo largo de este proyecto práctico, estaré trabajando los siguientes temas:

1. **Iniciando proyecto de Ansible** - Configuración inicial del entorno de trabajo
2. **Creando servidores con Ansible** - Automatización del aprovisionamiento de servidores
3. **Haciendo que los servidores cumplan órdenes** - Creación de playbooks y tareas
4. **Creando el archivo de inventario** - Definición de servidores cliente y sus grupos
5. **Configuración de servidores cliente** - Automatización de configuraciones específicas
6. **Poniendo todo junto** - Playbooks completos y orquestación de tareas
7. **Usando el método pull de Ansible** - Implementación de configuración autogestionada

## 📚 Sobre el Libro

**Mastering Ubuntu Server, Fourth Edition** de Jay LaCroix es una guía completa para administradores de sistemas que desean dominar Ubuntu Server en entornos empresariales, cubriendo desde la instalación básica hasta temas avanzados como automatización, contenedores y cloud computing.
